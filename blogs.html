Question-1: What are the different ways to select an element in the DOM?

In JavaScript we can select elements in the DOM(Document Object Model) using various methods. Here are the ways:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference Between innerHTML, innerText, and textContent in JavaScript

1. innerHTML

  • Returns the entire HTML inside an element.
  • Parses and executes any HTML tags inside the string.
  • We can use it when we want to modify including HTML tags

2. innerText

  • Returns only the text that is visible to the user.
  • Ignores all HTML tags inside the element.
  • Excludes hidden (display: none;) elements.

3. textContent

  • Returns all text content, including hidden elements.
  • Ignores all HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is to add event listener to the parent instead of adding event listeners to each individual child element. Inside the event handler, we use (event.target) to determine which child was clicked.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a concept where an event starts from the target element and propagates up through its parent elements. basically when an event occurs on the child element, then it propagate up to parent and then the parent's parent
